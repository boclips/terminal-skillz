## Move back / forward characters

C-b / C-f

## Move back / forward words

M-b / M-f

## Move to beginning of line

C-a

## Move to end of line

C-e

## Change directory

cd some/dir

## Change back to previous directory

cd -
(see also: git checkout -)

## Push a directory onto the stack

pushd some/dir

## Pop from the stack

popd
